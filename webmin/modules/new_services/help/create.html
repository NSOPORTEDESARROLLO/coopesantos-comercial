Create New Certificate
You can download it and install it on your local computer. Please fill all fields:

  • Certificate Name: Name of new Certificate, must be unique.
  • Common Name: You can use Server Name Here.
  • Country Code: Country code, Not use full country Name.
  • Street: Street Name.
  • Localitation: Localitation/Province/State.
  • Organitation: Organization name.
  • Email Address: Email address.
  • Valid For: Time for valid certificate in Years.
